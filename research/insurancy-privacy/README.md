# Health Insurance Data Sharing Practices and Privacy Practices

## Insurance Privacy Policy Analysis

A systematic document analysis of member-facing privacy policies from major private insurers 
operating in Georgia, Massachusetts, and Florida.

### Research Question

How do private insurer data collection and consent practices function for people with chronic 
conditions and system involvement histories, and what does meaningful consent actually require 
for this population?

### States

Georgia (primary), Massachusetts, Florida

### Methods

Deductive document analysis using three coding dimensions: consent mechanics, data scope, 
and readability. See individual README files in each subfolder for methodology details.
